# Tor Binary for Windows

> **Note:** The LOS Wallet app **auto-downloads** the Tor Expert Bundle on first launch.
> You do NOT need to manually place `tor.exe` here unless you want to use a custom build.

**This directory should contain:**
- `tor.exe` - Tor daemon for Windows

## How to obtain:
1. Download Tor Expert Bundle from https://www.torproject.org/download/tor/
2. Extract `tor.exe` from the bundle
3. Place it in this directory

## Alternative:
- Extract from Tor Browser Bundle for Windows
- Located at: `tor-browser/Browser/TorBrowser/Tor/tor.exe`

## Requirements:
- Windows 10 or later (64-bit)
- No additional installation needed
